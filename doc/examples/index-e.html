  • Overview
    To summarize fml installation (here paths is default value),
    • install fml to /usr/local/fml
    • create "elena" ML (elena ML HOME is /var/spool/ml/elena )
    • configure /var/spool/ml/elena/config.ph
    • setup elena ML configuration in /etc/aliases
    You can customize elena ML by configuring "config.ph".
