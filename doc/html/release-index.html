Mailing List に関して


fml について議論するML(自動的に登録するようになっている) fml-support@ffs.fml.org

参加する場合は、"subscribe Elena Lolabrigita" のように subscribe 名前 を書いて、
fml-support-ctl@ffs.fml.org に送って下さい。参加の意志を確認するメールが返ってくるのでそれにREPLY をして下さい。 質問、ご要望は ML に参加してもらって情報交換してもらうのがお互いのため に望ましいと考えています。
作者個人宛のDMでももちろん構わないのですが、そこで知りえた情報が他の人 に繁栄されないものなので…
また、作者が忙しくて答えられない場合も、MLであれば誰かが先に答えてくれ るかもしれません。
そういうわけですので、よろしければ、MLの方へ情報はお寄せください。

聞きたいことがあるが、MLに参加したくはない場合 fml-query@ffs.fml.org にだせば、作者とfml-support@ffs.fml.orgにフォワードし、fml-support ML への登録はしません。
こうすれば、MLの人たちにも情報（例：バグレポート）が伝わるし、 すぐに答が返ってくるかもしれません。

どうしても作者宛の DM で聞きたいということであれば fml-bugs@ffs.fml.org へ、お願いします。
