ftp hierarchy 


ftp://ftp.iij.ad.jp/pub/IIJ/dist/fukachan/fml/ の下の階層について

	current/		開発コード
	stable/			3.0リリースのバグつぶしシリーズ

current/ も stable/ も daily ではなくある程度区切りのついたところで僕
が作っています。そのためそれなりにちゃんと動くはずです。

	daily-snapshots/	"daily" snapshots
		current/	current branch の daily snapshots
		stable/		stable  branch の daily snapshots

daily なので中途半端なところでパッケージを生成視してしまって失敗してい
る可能性あり。おっかけむき。

	utils/			mail related softwares
	archives/		old snapshots for a study of history
