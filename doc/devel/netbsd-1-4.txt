	simple half-standard NetBSD/i386 installation


1) 	get boot.fs and prepare binaries if possible

ftp://ftp.iij.ad.jp/pub/NetBSD/NetBSD-1.4/i386/installation/floppies/boot.fs

(NetBSD/i386 1.4D ELF format version)
ftp://ftp.iij.ad.jp/pub/NetBSD/arch/i386/snapshot/07041999/


2)	make a boot floppy by dd boot.fs

# dd if=boot.fs of=/dev/rfd0a


3) 	insert boot floppy and start IBM-PC/PC clone machine.


4)	select "a" in the menu and installation.

	exit without *tgz installation

	check bootale but fails to exec /netbsd :)

	again 3), exit sysinst and goto 5)


5) 	set up ETHER-INTERFACE

# ifconfig -a
	... ...
fxp0:	... ...
	... ...
lo0:	... ...
	... ...

# ifconfig ETHTER-INTERFACE 192.168.0.100 netmask 255.255.255.0


6)	prepare /mnt/usr by mounting disks

# mount /dev/wd0a /mnt
# mount /dev/wd0e /mnt/usr
# mkdir /mnt/usr/INSTALL


7)	get all (prepared) *tgz from local machine and

# cd /mnt/usr/INSTALL
# ftp 192.168.0.3
	...
ftp> get *tgz
	...
ftp> quit


8) 	unmount disks

# umount /mnt/usr
# umount /mnt


9)	run /sysinst and select "c"

select "local" with the directory "/mnt/usr/INSTALL"

	installing ...


10)	reboot


11)	edit /etc/rc.conf
	set up password
