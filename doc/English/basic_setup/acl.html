Summary: What ML members can do and cannot do

By default
Who is a member of ML ? mail addresses contained in files defined in @MEMBER_LIST
members and members-admin by default
recipients of articles mail addresses contained in files defined in @ACTIVE_LIST
only actives by default

By default
who can post to ML a ML member
who can use fml commands a ML member

FAQ

Q: a remote administrator (who is not a member) can post ?
A: yes by default. Consider the following case

  • a remote administrator not a ML member. He/She do not read ML articles.
  • But the remote administrator sometimes need to forward ML articles to ML.
