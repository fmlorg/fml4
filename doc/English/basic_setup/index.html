  • How to make a ML. Consider elena ML case.

    Caution: we assume this operation is done on the host where fml ins installed.
    (See     ../GUI for fml CGI interface)

    • (Condition: MTA (sendmail,postfix,qmail,...) is already set up)
    • install fml to /usr/local/fml
    • make elena ML. (elena ML HOME directory is /var/spool/ml/elena )
    • add /var/spool/ml/elena/aliases to /etc/aliases (edit file). After you edit it, do not forget to run "newaliases". How to update aliases depends on your MTA.
    • add a ML member
    customize is to edit /var/spool/ml/elena/config.ph. Each ML has each config.ph
    /var/spool/ml/elena/site_force.ph and site_init.ph enforces the same configuration over the whole ML.
