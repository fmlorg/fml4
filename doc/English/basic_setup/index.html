  • ＭＬの作り方の概要: 例は elena ＭＬです。pathは適当に読みかえてください。

    注意: このページは fml をインストールしたマシンの上で行なう操作方法です。
    (GUIについては    ../GUIをみてね)

    • (前提条件: MTA (sendmail,postfix,qmail,...) の設定はすでに終っているはず)
    • /usr/local/fml に fml をインストールする
    • elena MLを作る (作られるディレクトリ: /var/spool/ml/elena )
    • /var/spool/ml/elena/aliases を /etc/aliases に付け加える(ファイルの編集)。 編集したら newaliases などを忘れずに (update の手順は使っているMTAに依存します)。
    • メンバーを登録する。
    カスタマイズ は設定ファイル(/var/spool/ml/elena/config.ph)で行ないます。 ＭＬごとに異なる設定です。
    設定ファイルのメインテナンス方法については松竹梅(?)あります。
    全ＭＬ共通の設定を強制することもできます (/var/spool/ml/elena/site_force.ph)。
