add a member to elena ML 
If you add nospam@fml.org as a member to elena ML, run the following
command. 

% /usr/local/fml/makefml add elena nospam@fml.org

Contrarly, remove nospam@fml.org, run this way.

% /usr/local/fml/makefml bye elena nospam@fml.org
