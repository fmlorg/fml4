メンバーと非メンバーでメールサイズの制限値を変えたい


●メンバーと非メンバーでメールサイズの制限値を変えたい 
【fml-support:09159】

(1) メンバー用のサイズを設定しておく(あるいは設定しない==無制限)

INSTALL.jp
| ○ MAX_MAIL_SIZE (default: 未定義 == 無制限)
|  メールサイズの最大値(デフォルトでは無制限)。貼付ファイルを許すか否か
|  でサイズ制限は決まるだろう。ちなみに fml.org では500K bytes にしている。

(2) メンバーでなかったら、動的に設定しなおす。
	$START_HOOK =q%
	   if (! &MailListMemberP($From_address)) {
		$MAX_MAIL_SIZE = 30 * 1024; # 30KB の例
	   }
	%;
