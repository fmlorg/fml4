UTF-8 (Unicode) のメールを JIS に変換して配送する


●UTF-8 (Unicode) のメールを JIS に変換して配送する 
【fml-support:08783】

#Jcode.pm のインストールが必要（要 perl5）
#
$START_HOOK = q`
    if ($Envelope{'h:content-type:'} =~ /utf-8/i) {
        use Jcode;
        &Jcode::convert(\$Envelope{'Body'}, 'jis', 'utf8');
        &DEFINE_FIELD_FORCED("content-type",
            'text/plain; charset="ISO-2022-JP"');
        &DEFINE_FIELD_FORCED("content-transfer-encoding", "7bit");
    }

`;
