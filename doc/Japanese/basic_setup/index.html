  • 概要: インストール手順の簡単なまとめ (以下pathは適当に読みかえてください)
    • /usr/local/fml にインストールする。もしくはすでにされている
    • elena MLを作ったら (ディレクトリは /var/spool/ml/elena )
    • /var/spool/ml/elena/config.ph の設定をする
    • /etc/aliases に設定/設定したことを確認する(MTA依存)
    でインストールは終りです。 カスタマイズは 設定ファイル config.phで行ないます。
