  • 概要: インストール手順の簡単なまとめ
    (以下pathは適当に読みかえてください)
    • /usr/local/fml にインストールする/されている
    • elena MLを作ったら (ディレクトリは /var/spool/ml/elena )
    • /var/spool/ml/elena/config.ph の設定をする
    • /etc/aliases に設定/設定したことを確認する(MTA依存)
    で手順のすべてです。 カスタマイズは 設定ファイル config.phで行ないます。
    • fmlのインストールの概観 　(詳細) 　[Q and A] 　[CGIのインストール (fml-current only)]
      注: fml 2.x から 3.0 へのアップグレード
    • Postfix のインストールの例 (NetBSD-currentのpackage system)
      最近のpostfixはmake installするとインストーラが 対話的に聞いてくれる。それに答えていけば 基本設定はそれだけで十分だ。
