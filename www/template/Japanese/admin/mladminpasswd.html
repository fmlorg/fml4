"fml" _FML_VERSION_ Configuration Interface

_ML_ メーリングリストの設定

管理者パスワードの設定 (_ML_ ＭＬ) ＭＬを選択して、管理者のパスワードを設定して下さい。 これは
  • admin と approve コマンドのパスワード
  • およびGUIのメニューに入る
時に使うパスワードです。 簡単化のためGUIメニューではこの３つに同じパスワードを設定します。
CUIの makefml htpasswd を用いることでコマンド用とCGI用と 別々のパスワードをつけることが可能です。
注意: 新しい管理者名を入力すると、管理者の新規登録扱いになります
